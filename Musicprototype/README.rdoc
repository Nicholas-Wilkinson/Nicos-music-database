== LEESMIJ
Versie 1.0
Dit is een test programma om een muziek database te beheren, je kunt hier me band en hun liedjes opslaan met een link naar een tableture van het liedje.
De search functie zoekt zowel naar bands als naar liedjes. Je kunt ook bands en liedjes toevoegen (maar nog niet verwijderen).


* Gebruikte Programma's
De Ruby versie die voor dit programmaatje is gebruikt is "Ruby 2.0.0p645" voor Windows 7 64-bit.
De Rails versie is "4.1.8".
